# Math Number Trainer

A simple mathematics practice game built with Python.

The user can practice addition and multiplication while the program checks answers automatically.

---

## Features

- Addition practice
- Multiplication practice
- Automatic answer checking
- Menu-driven interface

---

## Concepts Used

- Functions
- Return Values
- Conditional Statements
- User Input
- Program Organization

---

## Why I Built This

I created this project to practice writing reusable functions and separating different tasks into smaller parts instead of writing everything inside one large block of code.
